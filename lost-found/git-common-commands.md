# Git 常用命令

预览两个分支的差异
```shell
$ git diff <source_branch> <target_branch>
```

内建的图形化 git
```shell
$ gitk
```

显示历史记录时，每个提交的信息只显示一行（默认 medium）
```shell
$ git config format.pretty oneline
```
